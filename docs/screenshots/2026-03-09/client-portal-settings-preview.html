Como usar o acesso hibrido

Resumo operacional para a equipe.

  1. Manter a pagina do portal publicada com o shortcode oficial.
  2. Cadastrar um e-mail valido em cada cliente.
  3. Usar links para acesso imediato e senha para recorrencia.
  4. Sincronizar usuarios antigos quando o admin indicar pendencia.
