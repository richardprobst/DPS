# Registro de Evolução Visual — DPS Signature

Este arquivo registra a evolução visual do DPS.

Desde **17/04/2026**, a referência oficial para novas interfaces deixou de ser o padrão M3 e passou a ser o sistema proprietário **DPS Signature**, documentado em:

- `docs/visual/FRONTEND_DESIGN_INSTRUCTIONS.md`
- `docs/visual/VISUAL_STYLE_GUIDE.md`

## Como interpretar os registros antigos

Entradas anteriores a essa data podem mencionar `M3` ou `Material 3 Expressive`. Essas referências permanecem como contexto histórico de implementação e não como instrução ativa para novos trabalhos.

| Data | Marco | Status | Notas |

| 2026-04-17 | Transição oficial para `DPS Signature` | ✅ Concluído | Nova identidade visual proprietária consolidada a partir da linguagem aplicada nas páginas públicas do site. |

## Princípios ativos do DPS Signature

- base sóbria e premium;
- cantos predominantemente retos;
- tipografia `Sora + Manrope`;
- composição editorial;
- cor de acento fria, usada com contenção;
- verde reservado para ação real;
- contraste alto;
- menos ornamento, mais estrutura.
