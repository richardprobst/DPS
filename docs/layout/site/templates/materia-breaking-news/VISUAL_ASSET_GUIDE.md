# Guia de Apoio Visual - Matérias da Desi Pet Shower

Este documento define como escolher o melhor apoio visual para cada matéria.

Nem toda matéria precisa de foto. Nem toda matéria precisa de dado. O apoio visual deve existir quando realmente melhora a leitura.

## Regra principal

Escolha o apoio visual que mais ajuda o leitor a entender o assunto mais rápido.

Prioridade:
1. clareza
2. relevância
3. sobriedade
4. identidade visual

## Tipos de apoio visual

### 1. Foto editorial

Use quando a matéria ganha com:
- atmosfera;
- contexto humano;
- percepção de ambiente;
- proximidade emocional.

Boa para:
- bastidores;
- equipe;
- ambiente;
- experiência de atendimento;
- histórias leves e institucionais.

Evite foto quando:
- ela só estiver "decorando";
- o assunto pedir orientação prática;
- a imagem não acrescentar entendimento.

### 2. Quadro visual

Use quando a matéria precisa:
- organizar sinais;
- resumir pontos;
- comparar cenários;
- acelerar a leitura.

Boa para:
- checklists;
- "sinais de atenção";
- o que observar;
- o que evitar;
- passo a passo curto;
- resumo de leitura.

Este é o melhor recurso para a matéria piloto sobre sinais na rotina de banho.

### 3. Dado visual

Use quando a matéria precisa:
- mostrar comparação;
- resumir frequências;
- apresentar números;
- mostrar distribuição ou variação.

Boa para:
- comparações de rotina;
- tempos;
- frequência;
- volume;
- evolução.

Se não houver dado confiável, não invente gráfico.

## Decisão prática

Use esta regra:

- se o assunto for diagnóstico, orientação ou checklist:
  - use quadro visual
- se o assunto for história, atmosfera ou bastidor:
  - use foto editorial
- se o assunto for comparação numérica:
  - use dado visual

## Regra de sobriedade

O apoio visual:
- não deve parecer banner;
- não deve parecer post de rede social;
- não deve competir com o título;
- não deve usar excesso de cor;
- não deve quebrar o ritmo do texto.

## Aplicação no template

As matérias usam por padrão:
- `dps-editorial-figure`
  - container da área visual
- `dps-editorial-board`
  - para quadros visuais

Se a matéria pedir foto:
- a foto deve entrar dentro de `dps-editorial-figure`;
- a legenda deve explicar por que a imagem está ali;
- a imagem deve ser limpa, realista e sem poluição textual.

Se a matéria pedir dado visual:
- o gráfico deve ser simples;
- o título do quadro precisa dizer o que está sendo comparado;
- a leitura deve continuar possível mesmo sem interpretar o gráfico em detalhes.

## Regra para futuras matérias

Antes de criar qualquer apoio visual, responder:
1. isso ajuda a entender mais rápido?
2. isso é o melhor formato para este assunto?
3. isso mantém o tom editorial da Desi Pet Shower?

Se a resposta não for claramente "sim", não use.
