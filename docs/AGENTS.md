# Instruções para documentação

## Escopo
Este arquivo cobre todo o diretório `docs/`.

## Convenções de escrita
- Mantenha o conteúdo em português, salvo quando for necessário citar termos técnicos.
- Utilize frases curtas e objetivas para facilitar a leitura.
- Sempre que possível, inclua exemplos de uso para comandos ou fluxos descritos.

## Formatação
- Prefira títulos em estilo frase (apenas a primeira palavra em maiúscula).
- Use listas numeradas para sequências de passos e listas não ordenadas para coleções de itens.
- Evite linhas com mais de 100 caracteres.

## Organização
- Antes de criar um novo arquivo, verifique se já existe uma página relacionada e considere atualizá-la.
- Adicione entradas em índices ou sumários sempre que novos tópicos forem criados.
- Atualize referências cruzadas quando arquivos forem renomeados ou movidos.
